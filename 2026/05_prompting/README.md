### Lecture 5: Prompting

This folder contains two small demonstrations showing how the Jinja template engine can be used to manage prompt templates outside the main Python code. The examples illustrate how templating is useful when prompts contain variables, conditional logic, repeated structures, or task-specific instructions.

1. `sentiment-analysis/`  
   Demonstrates prompt templating for a classical NLP task: sentiment analysis.

2. `scientific-task-example/`  
   Demonstrates prompt templating for a scientific information-extraction task.
